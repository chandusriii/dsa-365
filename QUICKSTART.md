# Quick Start Guide 🚀

## 5-Minute Setup

### 1. Clone Repository
```bash
git clone https://github.com/chandusriii/dsa-365.git
cd dsa-365
```

### 2. Explore Structure
```bash
ls -la
# See: problems/, cheat-sheets/, progress/, company-wise/
```

### 3. Read Main Cheat Sheet
```bash
cat cheat-sheets/ultimate_dsa_cheat_sheet.md
```

### 4. Pick Your Path

**Path A: Learn by Topic**
```
Go to: problems/01-arrays/
Start with Easy problems
Work your way to Hard
```

**Path B: Company Interview Prep**
```
Go to: company-wise/google.md (or amazon, microsoft, etc.)
Solve in recommended order
```

**Path C: 365-Day Challenge**
```
Solve 1 problem daily
Update progress-tracker.md
Build a streak
```

## Daily Routine (30 minutes)

```
1. Read problem (5 min)
2. Try solving (15 min)
3. Check solution (5 min)
4. Code from scratch (5 min)
```

## Track Progress

Update `progress/progress-tracker.md` daily:

```markdown
### Day 1
- Problem: Two Sum
- Difficulty: Easy
- Status: ✅ Solved
- Time: 25 min
- Confidence: 7/10
```
