# Striver A2Z DSA Sheet Guide

A practical guide to use this repo together with Striver's A2Z DSA Sheet.

## Main Link
- Striver A2Z Sheet: https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-sheet-2

## Why Use It
- Structured progression from basics to advanced topics.
- Strong interview-focused problem curation.
- Good companion videos and explanations.

## Recommended Flow
1. Pick one topic from this repo (for example `problems/01-arrays`).
2. Solve one problem yourself first (20 to 30 minutes).
3. Compare with this repo solution.
4. If still unclear, review Striver explanation/video.
5. Re-implement from scratch without looking.
6. Update `progress/progress-tracker.md`.

## Pro Tips

- Always write complexity before finalizing.
- Keep a short note for each mistake pattern.
